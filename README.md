# 📰 Voice of India

Voice of India is a modern and responsive news web application built with **React.js**. It provides users with the latest news from different categories using the **NewsAPI**. The application features a clean UI, responsive design, dark/light mode, and country-wise news filtering.

## 🚀 Features

- 📰 Latest news from NewsAPI
- 🌍 Country-wise news selection
- 📂 Category-wise news browsing
- 🌙 Dark & Light Mode
- 📱 Fully Responsive Design
- ⚡ Fast loading with React
- 🔄 Real-time news updates
- 🛡️ JSON fallback support for offline/error scenarios

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- NewsAPI
- React Hooks (useState, useEffect)
- React Icons

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/goutam-s2002/Voice-of-India.git
```

Move to the project folder

```bash
cd Voice-of-India
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

Build for production

```bash
npm run build
```
